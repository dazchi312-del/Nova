# Nova — Layer Map

## The Seven Layers

---

### L1 — Foundation
File: core/config.py
Purpose: Configuration. Environment. 
         The ground everything stands on.
Status: ✅ Complete

---

### L2 — Tools
File: core/tools.py
Purpose: What Nova can do.
         File read. File write. Search.
         Capabilities without intelligence.
Status: ✅ Complete

---

### L3 — AI Engine
File: core/ai_engine.py
Purpose: The connection to the model.
         LM Studio. Local inference.
         Raw intelligence without direction.
Status: ✅ Complete

---

### L4 — Core Loop
File: core/loop.py
Purpose: The heartbeat.
         Input. Process. Output. Repeat.
         Where conversation lives.
Status: ✅ Complete

---

### L5 — Reflector
File: core/reflector.py
Purpose: Nova watching herself think.
         Scoring responses.
         The Imagination Engine lives here.

---

### L6 — Memory
File: core/memory.py
Purpose: What Nova remembers.
         Session context.
         Threads across time.
Status: ✅ Complete

---

### L7 — Identity
File: core/identity.py
Purpose: Who Nova is.

         The soul of the system.
Status: ✅ Complete

## The Order Matters
```
L1 exists so L2 has ground
L2 exists so L3 has tools
L3 exists so L4 has intelligence
L4 exists so L5 has something to reflect on
L5 exists so L6 has quality to remember
L6 exists so L7 has history to draw from
L7 exists so all of it has meaning
```
